About Me

Born and raised in California, traveled the world and now back where I started. Having grown up in Silicon Valley, I was always intrigued by technology but never brave enough to fully explore that field. Now having completed a fullstack MERN web development boot camp at George Washington University, I wish I had started a long time ago.

I remember my first computer, it was a Windows '99 machine that ran on dial-up, playing video games on that thing was like watching PowerPoint. In middle school our computers got viruses because we were using Internet Explorer and I know now that IE connected the internet to the user's file system which is a glaring security issue, but when I was a kid I wondered why. These early interactions with computers sparked my interest in learning more about them. Why is my computer so slow? How did all of these malicious viruses get there? It wasn't until in my early twenties when I recognized this passion for tech and only this past year that I started to explore this newly discovered love affair. After much consideration and self study, I eventually applied to attend a coding boot camp in hopes of changing careers. Since attending George Washington's Fullstack Web Developer Coding Boot camp, I have fallen in love with programming. There are so many possibilities and different solutions to the same problem; this fascinates me and I continue to grow as a developer everyday as well as my yearning to know more. I have truly found my passion.
